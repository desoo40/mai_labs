Выбранный пункт отметить * справа от номера пункта.
В каждом разделе можно выбрать только один пункт.

1. Вычитание.
	1.1 Один и более операндов.
	 (- x y z)
	1.2 “олько два операнда.
	 (- x y)
	1.3* Только один операнд.
	 (- x)

2. Деление.
	2.1 Один и более операндов.
	 (/ x y z)
	2.2* Только два операнда.
	 (/ x y)
	2.3 Только один операнд.
	 (/ x)

3. Числовые литералы токена $zero.
	3.1* В общем контексте числового выражения.
	 0

4. Числовые литералы токенов $int и $float.
	4.1 В общем контексте числового выражения.
	 (+ 1 1)
	4.2* Только в определении переменной.
	 (define one 1)(+ one one)
	4.3 Только в определении процедуры.
	 (define (one) 1)(+ (one) (one))
	4.4 Только в определении локальной переменной.
	 (define (one) (define one 1)one)(one)

5. Оператор OR.
	5.1 Один и более операндов.
	 (or #t #f #f)
	5.2 Только два операнда.
	 (or #t #f)
	5.3* Отсутствует.

6. Оператор AND.
	6.1* Один и более операндов.
	 (and #t #f #f)
	6.2 Только два операнда.
	 (and #t #f)
	6.3 Отсутствует.

7. Оператор  = .
	7.1* Есть.
	 (=  x y)
	7.2 Отсутствует.

8. Оператор отношения, кроме оператора = .
	8.1* (<  x y)
	8.2 (<= x y)
	8.3 (>  x y)
	8.4 (>= x y)

9. Оператор IF для чисел.
	9.1 Есть.
	 (if #t e pi)
	9.2* Отсутствует.

10. Оператор IF для строк.
	10.1* Есть.
	 (display(if (p?)"Yes""No"))
	10.2 Отсутствует.

11. Оператор COND.
	11.1 Составная ветвь ELSE, ноль и более составных клауз.
	 (cond(else(display pi)pi))
	11.2 Составная ветвь ELSE, одна составная клауза.
	 (cond((e?)(display e)e)(else(display 0)0))
	11.3* Простая ветвь ELSE, ноль и более простых клауз.
	 (cond(else 0))
	11.4 Простая ветвь ELSE, одна проста€ клауза.
	 (cond(#f e)(else pi))
	11.5 Без ветви ELSE, одна и более составных клауз.
	 (cond(#t(display pi)pi))
	11.6 Без ветви ELSE, две составные клаузы.
	 (cond((e?)(display e)e)(#t(display 0)0))
	11.7 Без ветви ELSE, одна и более простых клауз.
	 (cond(#t pi))
	11.8 Без ветви ELSE, две простые клаузы.
	 (cond(#f e)(#t pi))
	11.9 Отсутствует.

12. Определяемые переменные.
	12.1 Глобальные и локальные.
	 (define a 1)(define(f)(define a 2)a)
	12.2* Только глобальные.
	 (define a 1)(define(f)a)
	12.3 Только локальные.
	 (define(f)(define a 2)a)
	12.4 Отсутствуют.
	 (define(f a)(set! a pi)a)(f e)
